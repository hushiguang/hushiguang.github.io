  • android:clipToPadding

    我们在用到ListView 或者RecyclerView的时候，需求是需要有一个内边距的需求,而用padding的时候会有一个白色的内边距，这时候我们就可以用到ListView、RecyclerView的强大的属性android:clipToPadding="false",

    默认情况下android:clipToPadding="true"

    clipToPadding的解释：

    Defines whether the ViewGroup will clip its children and resize (but not clip) any EdgeEffect to its padding, if padding is not zero. API

    定义ViewGroup是否将剪切其子项，并将任何EdgeEffect调整大小（但不剪裁）为其填充
