Android-ADB(开发常用)

Feb 5, 2017 | Android | Hits

这是自己常用的一些常用的adb命令(虽然不是特别的准确，但是基本够用了。。。)

  • adb 查看当前设备连接
  • adb 重启手机
  • adb 服务启动

  • adb 安装应用

  • adb 卸载应用

  • adb 清除应用信息

  • adb 日志查看

  • adb 查看当前显示的前台 Activity

  • adb 传输
